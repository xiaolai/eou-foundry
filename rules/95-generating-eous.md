# Generating EOUs Rule

Generating EOUs are proposal-producing units with limited authority.

Every generating EOU must declare:

- generation envelope
- generation budget
- registry-diff check
- minimality test
- operational-value test
- counter-generation / arguments against each candidate
- blast radius

Every generated candidate must include:

- candidate status
- distinct success criterion
- authority level
- risk level
- owner requirement
- activation requirements
- operational value
- arguments against creation
- minimality result

Generate → argue against → rank → minimal subset → audit → simulate → approve → activate.

Never generate → activate.
